# Lista Doblemente Enlazada - Java

Implementación de una Lista Doblemente Enlazada en Java con menú interactivo en consola.

## Diagrama de Clases

![Diagrama de Clases](https://drive.google.com/file/d/1lGx5L0McJ7pRfGiMRSB6THPaPsRRzVCi/view?usp=sharing)

## Descripción

Programa desarrollado en Java que implementa la estructura de datos Lista Doblemente Enlazada. Cada nodo contiene un dato entero y referencias al nodo anterior y siguiente.

## Clases

- **Nodo**: Representa cada elemento de la lista con sus referencias anterior y siguiente.
- **ListaDoble**: Contiene toda la lógica de la estructura de datos.
- **Main**: Punto de entrada del programa con el menú interactivo.

## Funcionalidades

1. Insertar al inicio
2. Insertar al final
3. Recorrer hacia adelante
4. Recorrer hacia atrás
5. Mostrar tamaño de la lista
6. Mostrar si la lista está vacía
7. Buscar elemento por valor
8. Buscar elemento por índice
9. Borrar un elemento
